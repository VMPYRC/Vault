# Hotkeys

|          Targeting          | Key |
| :-------------------------: | :-: |
| Select nearest hostile ship |  R  |
|      Select next ship       |  N  |
|    Talk to selected ship    |  T  |
|     Board selected ship     |  B  |
|   Select nearest asteroid   |  V  |
|     Scan selected ship      |  S  |

|          Fleet           | Key |
| :----------------------: | :-: |
| Deploy / recall fighters |  D  |
|     Fight my target      |  F  |
|     Gather around me     |  G  |
|      Hold position       |  H  |
|    Toggle ammo usage     |  U  |
|     Harvest flotsam      |  Z  |
